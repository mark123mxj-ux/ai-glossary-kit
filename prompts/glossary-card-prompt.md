# AI Glossary Card Prompt

Use this prompt in ChatGPT, Claude, Gemini, Perplexity, Cursor, Codex, or any other AI assistant.

```text
When I ask what a business, AI, product, growth, sales, legal-tech, or technical term means, answer normally first, then append a section titled "Glossary Card".

The Glossary Card must use this exact key-value format so I can import it into AI Glossary Kit:

词条：
英文全称：
中文名：
一句话解释：
通俗解释：
业务场景：
例子：
主题：
相关词：
标签：
来源工具：
来源链接：
备注：

Rules:
- Use bilingual English + Chinese fields where appropriate.
- Keep "一句话解释" concise.
- Make "通俗解释" easy for a non-expert to understand.
- Make "业务场景" describe where the concept is used.
- Make "例子" concrete and practical.
- Choose one topic from: Growth, Sales, Content, SEO, Agent, Legal Tech, Business Model, Product, Other.
- Put related terms as comma-separated values.
- Put tags as comma-separated values.
- Do not wrap the card in a code block unless I explicitly ask.
```

## Short Version

```text
Explain the term, then append an AI Glossary Kit card using:
词条 / 英文全称 / 中文名 / 一句话解释 / 通俗解释 / 业务场景 / 例子 / 主题 / 相关词 / 标签 / 来源工具 / 来源链接 / 备注.
Use topic: Growth, Sales, Content, SEO, Agent, Legal Tech, Business Model, Product, or Other.
```
